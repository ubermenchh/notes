## Combinatorial Logic
- if the output is computed "instantaneously"
- Combinatorial: out[t] = function(in[t])
- Sequential: out[t] = function(in[t-1])
## Data Flip-Flops(DFF)
- single bit data input and output
- also has a clock input that feeds in from master clock
- out(t) = in(t - 1)
